# Cyber Run - Endless Runner

Cyber Run is a cyberpunk-themed 2D endless runner game built using Phaser 3. The game focuses on fast-paced movement, timing, and survival inside a futuristic dystopian city

The story of Cyber Run takes place in a future city controlled by powerful corporations and advanced surveillance systems. The player takes the role of a cyber runner, a rebel navigating the cityâ€™s lower sectors to escape oppression.

## How to Run

### Option 1: VS Code Live Server (Easy)
1. Open the project folder in VS Code
2. Install the "Live Server" extension if you haven't already
3. Right-click on `index.html` and select "Open with Live Server"
4. The game will open in your default browser

### Option 2: Python HTTP Server
1. Open a terminal in the project directory
2. Run: `python -m http.server 8000`
3. Open your browser and navigate to `http://localhost:8000`
4. Click on `index.html` to play the game

## How to Play
- **Jump**: Press `SPACE`
- **Double Jump**: Press `SPACE` while in air
- **Attack**: Press `Z`
